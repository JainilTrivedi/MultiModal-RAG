# MultiModel RAG on research papers


To run this project you would need API keys from AWS, Nvidia nim, HuggingFace.

- Create an AWS account and set up the access and secret key. Add them as `AWS_SECRET_KEY`, `AWS_ACCESS_KEY` to the secrets section of the colab.

- Create a HuggingFace account and setup a access token. Add it your secret section with the name `HF_TOKEN`.

- Create an account in nvidia nim, go to the `llama-3.2-90b-vision-instruct` model click on build this with nim and generate API key. Add it to the secrets with the name  `NVIDIA_API_KEY`.
